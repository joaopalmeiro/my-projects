## Development

Install [fnm](https://github.com/Schniz/fnm) (if necessary).

```bash
cp .env.local.example .env.local
```

```bash
fnm install && fnm use && node --version && npm --version
```

```bash
npm install
```

```bash
npm run dev
```

```bash
npm run lint
```

```bash
npm run format
```

## Deployment

```bash
npm run build
```
